# T-Библиотека

Простое консольное приложение для управления личной коллекцией книг. Делал для удобного учета прочитанного и того, что планируется к прочтению.

## Что умеет
- Добавление книг с описанием, автором, жанром и годом.
- Просмотр всей библиотеки: можно сортировать (по названию/автору/году) и фильтровать (по жанру или статусу прочитанности).
- Отметка прочитанного и добавление в избранное.
- Умный список избранного: прога анализирует жанры прочитанных и любимых книг и советует, что почитать дальше из нетронутого.
- Поиск по ключевым словам (ищет в названии, авторе и описании).
- Полное сохранение стейта. Данные пишутся в `library_data.json` и подтягиваются при следующем запуске.

## Запуск
Скрипт написан на чистом питоне, никакие внешние зависимости не нужны.

1. Клонируем репу или качаем файл `main.py`.
2. Запускаем через терминал:
   ```bash
   python main.py
